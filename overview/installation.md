# ⬇ Installation

```php
composer require fastvolt/framework
```
✨ **Requirements**:
- [ ] PHP 8.1
- [ ] Mysql (only if needed).
<br>

### 🖥️ Starting a server

```cmd
php -S localhost:8000 -t FastVolt/
```
> **NOTE**: `FastVolt` should be the directory name or folder name where foster framework was installed into.
